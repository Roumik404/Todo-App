<h2>📌 About The Project</h2>

<p>
TaskMaster Pro is a sleek and advanced To-Do & Productivity Web Application designed to help users stay organized, focused, and productive.
It combines task management, productivity analytics, Pomodoro focus sessions, streak tracking, and a modern dashboard UI into one seamless experience.
</p>

---

<h2>✨ Features</h2>

<ul>
  <li>📋 Create, Edit, Complete & Delete Tasks</li>
  <li>🎯 Priority-Based Task Management</li>
  <li>📂 Category Filtering & Sorting</li>
  <li>📅 Due Date Support</li>
  <li>🔍 Real-Time Search & Filters</li>
  <li>📊 Interactive Analytics Dashboard</li>
  <li>🔥 Productivity Streak Tracking</li>
  <li>⏳ Built-in Pomodoro Timer</li>
  <li>🌙 Dark / Light Theme Toggle</li>
  <li>💾 Local Storage Data Persistence</li>
  <li>📤 Export Tasks & Productivity Data</li>
  <li>📱 Fully Responsive Design</li>
  <li>🎨 Smooth Animations & Premium UI</li>
</ul>
